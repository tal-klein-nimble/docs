## What is Nimble Residential Proxy?

Nimble's BackConnect Gateway provides a single point of access to a global network of premium residential IPs. Built for developers who need reliable, high-performance proxy infrastructure without the complexity of managing IP lists.

## How It Works

<Steps>
  <Step title="Connect to Gateway">
    Use a single BackConnect proxy endpoint - no IP lists to manage
  </Step>
  <Step title="Authenticate">
    Choose username/password authentication or IP allowlist
  </Step>
  <Step title="Configure Targeting">
    Add optional parameters for geolocation and session control
  </Step>
  <Step title="Make Requests">
    Send standard HTTP requests through the proxy gateway
  </Step>
</Steps>

## Gateway Details

- **Proxy Host**: `ip.nimbleway.com`
- **Port**: `7000`
- **Protocol**: HTTP/HTTPS
- **Authentication**: Username/password or IP allowlist
