# User Endpoints

Manage user accounts, authentication, and account settings.

## Get User Profile

Retrieve current user account information.

```http
GET /user/profile
```

### Response

```json
{
  "id": "user_123",
  "email": "user@example.com",
  "plan": "premium",
  "usage": {
    "requests_used": 1500,
    "requests_limit": 10000
  }
}
```

## Update User Settings

Update user account preferences and settings.

```http
POST /user/settings
```

### Request Body

```json
{
  "notifications": true,
  "timezone": "UTC",
  "default_format": "json"
}
```

### Response

```json
{
  "success": true,
  "message": "Settings updated successfully"
}
```
