  ### Release Notes for Week Ending January 15, 2026

  ## 

  ### <Badge color="green" shape="rounded">âœ¨ New Features</Badge>

  #### Nimble SDK Python & Node.js

  - **Unified Web Tools SDK**: Launched comprehensive Python and Node.js SDKs providing seamless access to all Nimble Web Tools through a single interface. Install with `pip install nimble-py` or `npm install @nimbleway/sdk` and access Extract, Search, Map, Crawl, and Template features with consistent, intuitive methods.
  - **Extract API**: New `nimble.extract()` method retrieves clean HTML, text, and structured data from any webpage. Supports JavaScript rendering, stealth mode for anti-bot protection, custom parsing with CSS selectors, browser actions, screenshots, and network request capture. [Learn more](/documentation/web-tools/extract)
  - **Extract Templates API**: Pre-built, production-ready extractors for 45+ popular platforms including Amazon, LinkedIn, Google, and more via `nimble.extract_template()`. Zero configuration required - just provide params like product IDs or search terms. Templates are maintained 24/7 by Nimble and automatically updated when sites change. [Learn more](/documentation/web-tools/extract-templates)
  - **Search API**: Perform web searches and retrieve parsed content from top results with `nimble.search()`. Supports fast mode for URL discovery, deep search mode with full content extraction, and AI-powered answer summaries from search results. [Learn more](/documentation/web-tools/search)
  - **Map API**: Fast URL discovery and site structure mapping through `nimble.map()`. Ideal for discovering all pages on a website, extracting sitemap URLs, and understanding site architecture without deep crawling overhead. Supports subdomain filtering and exact domain matching. [Learn more](/documentation/web-tools/map)
  - **Crawl API**: Deep website crawling with `nimble.crawl()` for comprehensive data extraction across multiple pages. Features smart navigation, data normalization, and multi-page scraping with automatic pagination handling. [Learn more](/documentation/web-tools/crawl)
  - **Residential Proxy Access**: Simple proxy connection format through `gateway.nimbleway.com:7000` with support for geotargeting (country, state, city), sticky sessions, and geo-sessions. Access premium residential IPs from 195+ countries with automatic rotation or session persistence. [Learn more](/documentation/web-tools/proxy)
