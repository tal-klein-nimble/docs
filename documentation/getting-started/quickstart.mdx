# Quickstart Guide

This guide will help you make your first request with Nimble in just a few minutes.

## 1. Get Your API Key

First, obtain your API key from the [Nimble dashboard](https://online.nimbleway.com).

## 2. Installation

Install the Nimble SDK in your preferred language:

<CodeGroup>

```bash Python
pip install nimble-py
```


```bash Node.js
npm install @nimbleway/sdk
```

</CodeGroup>

## 3. Quick Examples

### Extract

Get clean HTML and structured data from any URL:

<CodeGroup>

```python Python
from nimble import Nimble

nimble = Nimble(api_key="YOUR-API-KEY")

result = nimble.extract({
    "url": "https://www.example.com",
    "render": True
})

print(result)
```


```node Node.js
import { Nimble } from "@nimbleway/sdk";

const nimble = new Nimble({ apiKey: "YOUR-API-KEY" });

const result = await nimble.extract({
  url: "https://www.example.com",
  render: true
});

console.log(result);
```


```bash cURL
curl -X POST 'https://gateway.nimbleway.com/api/v1/extract' \
--header 'Authorization: Bearer <YOUR-API-KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "url": "https://www.example.com",
    "render": true
}'
```

</CodeGroup>

### Extract Templates

Use pre-built templates for popular platforms:

<CodeGroup>

```python Python
from nimble import Nimble

nimble = Nimble(api_key="YOUR-API-KEY")

result = nimble.extract_template({
    "template": "amazon_pdp",
    "params": {
        "asin": "B08N5WRWNW",
        "zipcode": "90210"
    }
})

print(result)
```


```node Node.js
import { Nimble } from "@nimbleway/sdk";

const nimble = new Nimble({ apiKey: "YOUR-API-KEY" });

const result = await nimble.extractTemplate({
  template: "amazon_pdp",
  params: {
    asin: "B08N5WRWNW",
    zipcode: "90210"
  }
});

console.log(result);
```


```bash cURL
curl -X POST 'https://gateway.nimbleway.com/api/v1/extract-template' \
--header 'Authorization: Bearer <YOUR-API-KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "template": "amazon_pdp",
    "params": {
        "asin": "B08N5WRWNW",
        "zipcode": "90210"
    }
}'
```

</CodeGroup>
