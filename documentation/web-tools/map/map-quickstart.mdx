Nimble **Map** quickly discovers and maps all URLs from a website. It uses sitemaps and page analysis to provide fast URL discovery and site structure mapping, perfect for pre-crawl planning and quick site audits.

<Note>
  For comprehensive details, parameters, and advanced features, see the **Map Usage** page.
</Note>

## Mapping Quick Start

### Map API Endpoint

```bash
POST https://gateway.nimbleway.com/api/v1/map
```

### Installation

<CodeGroup>

```bash Python
pip install nimble-py
```


```bash Node.js
npm install @nimbleway/sdk
```

</CodeGroup>

### Basic Usage

Discover and map all URLs from a website using `nimble.map()`:

<CodeGroup>

```python Python
from nimble import Nimble

nimble = Nimble(api_key="YOUR-API-KEY")

result = nimble.map({
    "URL": "https://www.example.com"
})

print(f"Found {len(result['urls'])} URLs")
for url in result['urls']:
    print(url)
```


```node Node.js
import { Nimble } from "@nimbleway/sdk";

const nimble = new Nimble({ apiKey: "YOUR-API-KEY" });

const result = await nimble.map({
  URL: "https://www.example.com"
});

console.log(`Found ${result.urls.length} URLs`);
result.urls.forEach(url => console.log(url));
```


```bash cURL
curl -X POST 'https://gateway.nimbleway.com/api/v1/map' \
--header 'Authorization: Bearer <YOUR-API-KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "URL": "https://www.example.com"
}'
```

</CodeGroup>

### Response

```json
{
  "urls": [
    "https://www.example.com",
    "https://www.example.com/about",
    "https://www.example.com/products",
    "https://www.example.com/contact",
    "https://www.example.com/blog"
  ]
}
```

## Common patterns

### Fast sitemap mapping

Use only the sitemap for fastest results:

<CodeGroup>

```python Python
result = nimble.map({
    "URL": "https://www.example.com",
    "Sitemap": "only"
})

print(f"Found {len(result['urls'])} URLs in sitemap")
```


```node Node.js
const result = await nimble.map({
  URL: "https://www.example.com",
  Sitemap: "only"
});

console.log(`Found ${result.urls.length} URLs in sitemap`);
```


```bash cURL
curl -X POST 'https://gateway.nimbleway.com/api/v1/map' \
--header 'Authorization: Bearer <YOUR-API-KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "URL": "https://www.example.com",
    "Sitemap": "only"
}'
```

</CodeGroup>

### Include subdomains

Map URLs across all subdomains:

<CodeGroup>

```python Python
result = nimble.map({
    "URL": "https://www.example.com",
    "domainFilter": "subdomain"
})

print(f"Found {len(result['urls'])} URLs")
```


```node Node.js
const result = await nimble.map({
  URL: "https://www.example.com",
  domainFilter: "subdomain"
});

console.log(`Found ${result.urls.length} URLs`);
```


```bash cURL
curl -X POST 'https://gateway.nimbleway.com/api/v1/map' \
--header 'Authorization: Bearer <YOUR-API-KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "URL": "https://www.example.com",
    "domainFilter": "subdomain"
}'
```

</CodeGroup>

### Exact domain only

Restrict mapping to a specific domain:

<CodeGroup>

```python Python
result = nimble.map({
    "URL": "https://blog.example.com",
    "domainFilter": "exact domain"
})

print(f"Mapped {len(result['urls'])} URLs from blog subdomain")
```


```node Node.js
const result = await nimble.map({
  URL: "https://blog.example.com",
  domainFilter: "exact domain"
});

console.log(`Mapped ${result.urls.length} URLs from blog subdomain`);
```


```bash cURL
curl -X POST 'https://gateway.nimbleway.com/api/v1/map' \
--header 'Authorization: Bearer <YOUR-API-KEY>' \
--header 'Content-Type: application/json' \
--data-raw '{
    "URL": "https://blog.example.com",
```
