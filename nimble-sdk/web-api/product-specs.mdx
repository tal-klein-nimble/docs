# API Product Specifications

Detailed technical specifications for Nimble Web API products and capabilities.

## Performance Specifications

### Request Limits
- **Rate limits**: Up to 1000 concurrent requests
- **Timeout**: 180 seconds default (configurable)
- **Response time**: Average 2-5 seconds
- **Success rate**: 99.5% average success rate

### Data Formats
- **Response formats**: JSON, HTML, XML
- **Encoding**: UTF-8 support
- **Max response size**: 50MB per request
- **Compression**: Gzip compression supported

- **200+ locations** across 6 continents

### Supported Countries
- North America: US, Canada, Mexico

## Browser Support

### JavaScript Rendering
- **Chromium-based** rendering engine
- **Full DOM** access and manipulation
- **Modern web standards** support
- **Mobile rendering** capabilities

### Page Interaction
- **Click events** - Button clicks, form submissions
- **Keyboard input** - Text entry, keyboard shortcuts  
- **Scrolling** - Infinite scroll, lazy loading
- **Wait conditions** - Element visibility, network idle

## Security & Compliance

### Data Protection
- **SSL/TLS encryption** for all communications
- **No data retention** - Responses not stored
- **GDPR compliant** data processing
- **SOC 2 Type II** certified infrastructure

### Stealth Features
- **Real browser fingerprints** - Undetectable automation
- **IP rotation** - Automatic proxy switching
- **User-agent rotation** - Realistic browser profiles
- **Cookie management** - Session persistence

## API Endpoints

### Core Endpoints
- `POST /v1/scrape` - Real-time scraping
- `POST /v1/batch` - Batch processing
- `GET /v1/status` - Job status checking

### Specialized Endpoints

- `POST /v1/maps` - Google Maps data
