# Nimble SDK

Build powerful data collection workflows with Nimble's comprehensive SDK and APIs.

## What's Included

- **Web API** - General web scraping and data collection
- **Proxy API** - High-performance proxy network for stealth browsing
- **Client Libraries** - SDKs for popular programming languages
- **Specialized APIs** - SERP, Maps, and E-commerce endpoints

## Core Capabilities

- **Real-time data collection** from any website
- **Advanced stealth technology** to bypass detection
- **AI-powered parsing** to extract structured data
- **Global proxy network** with geo-targeting
- **Scalable infrastructure** for high-volume requests

## Getting Started

1. **Choose your approach:**
   - [Web API](/nimble-sdk/web-api/quickstart) - Direct HTTP API calls
   - [Proxy API](/nimble-sdk/proxy-api/quickstart) - Use with existing tools

2. **Set up authentication** with your Nimble credentials

3. **Start collecting data** with your first request
